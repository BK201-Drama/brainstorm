## 🎯 面向普通中年人的 Skills（推荐）

这些skill专门产出**简单易懂、接地气**的内容，适合没多少文化的中年人：

- **wechat-writing** - 微信公众号写作（大白话风格，最推荐⭐）
- **plain-language** - 内容人性化（去掉AI味，改成人话）
- **simple-content** - 内容生产（可以要求用简单话）
- **copywriting** - 文案写作（可以要求简化）
- **social-content** - 社交媒体内容（短视频脚本、朋友圈）

详细说明见：`面向普通中年人的内容技能.md`

### 6. Plain Language（内容人性化）⭐ 面向普通中年人
- **文件**: `plain-language/SKILL.md`
- **名称**: content-humanizer
- **描述**: 将AI生成的内容改得更像人写的，去掉AI味，用大白话
- **触发词**:
  - "this sounds like AI"
  - "make it more human"
  - "去掉AI味"
  - "改成人话"
  - "用大白话"

- **赚钱方式**:

  - 去AI味服务（10-50元/篇）
- **特点**: 专门去掉AI生成内容的机械感，改成人话

### 7. Simple Content（内容生产）⭐ 面向普通中年人

## 使用建议

### 内容创作流程（面向普通中年人）

1. **内容生成阶段**: 使用 `simple-content` 或 `wechat-writing`
   - 要求：用大白话、避免专业术语、面向普通人
   - 输出：初稿内容

2. **内容优化阶段**: 使用 `plain-language`
   - 去掉AI味，改得更像人话
   - 用生活化例子替换抽象概念

3. **文案优化阶段**: 使用 `copywriting` 或 `social-content`
   - 朋友圈文案：用 `copywriting`
   - 短视频脚本：用 `social-content`

### 创业规划流程

1. **需求发现阶段**: 使用 `brainstorm` skill
   - 触发: `$start`
   - 输出: PRD文档

2. **市场调研阶段**: 使用 `market-research`
   - 了解整体市场
   - 找到目标人群
